# 物流管理专业教学仿真系统

## —— 让学生"在系统里跑通一家物流公司"

---

### 一、项目一句话定位

**国内首款面向本科物流管理专业、覆盖企业全业务链的在线教学仿真系统**——学生以角色扮演的方式，在浏览器中完成从采购、运输、仓储到财务结算的全部物流业务操作，教师通过智能教学后台实时监控、自动评分、注入突发事件，实现"教、学、练、考"一体化。

---

### 二、为什么要做这个系统？

| 传统教学的痛点 | 本系统的解决方案 |
|:---:|:---:|
| 理论多、实操少，学生毕业后面对真实系统无从下手 | 仿真真实物流企业ERP，学生上手即练，零门槛 |
| 物流流程跨部门协作，课堂上无法模拟"多人协同" | 支持5-6人组队协作，每人扮演不同角色，实时联动 |
| 老师难以评估每位学生的实操水平 | 自动评分引擎，多维度量化考核（正确性+完整性+效率） |
| 企业场景千变万化，教材案例单一且陈旧 | 教师可自定义教学场景，可随时注入"突发事件"考验应变力 |
| 现有商业ERP太复杂、太贵，不适合教学 | 专为教学设计，轻量化、零安装、浏览器直接访问 |

---

### 三、系统能做什么？（核心功能一览）

#### 1. 全业务链仿真——六大核心模块

```
采购管理          运输管理          仓储管理
（申请→审批→     （下单→审核→     （入库→验收→
  订单→验收）      调度→跟踪→       上架→拣货→
                   签收）            发货→盘点）

         ↕                ↕                ↕
     库存管理          财务结算          合同管理
   （预警·流水）    （应付·应收）     （审批·终止）
```

每个模块都不是"摆设"，而是**完整可走通的业务链路**。例如采购管理：学生创建采购申请 → 提交审批 → 审批通过后生成订单 → 供应商发货 → 到货验收 → 系统自动生成入库单并更新库存——和真实企业流程一致。

#### 2. 多人实时协作——课堂就是"物流公司"

学生5-6人一组，分别扮演：

| 角色 | 职责 |
|:---:|:---|
| 采购经理 | 采购申请、供应商管理 |
| 仓库主管 | 入库验收、库存盘点 |
| 调度专员 | 车辆调度、司机分配 |
| 运输主管 | 运输跟踪、签收确认 |
| 财务专员 | 应收应付、结算对账 |
| 客户经理 | 客户管理、合同签订 |

所有操作通过 **WebSocket 实时同步**，一个人提交了采购审批，仓库主管立刻收到待办通知——课堂瞬间变成一个"小型物流公司"。

#### 3. 智能教学后台——教师的全景指挥中心

- **场景管理**：预设5种典型物流场景（日常运营、旺季爆发、应急物资等），教师也可自定义
- **实时进度监控**：教师大屏实时查看各组进度，谁卡在哪一步一目了然
- **突发事件注入**：教师可随时注入6种突发事件（车辆故障、客户退货、仓库爆仓等），测试学生应变能力
- **自动评分引擎**：按模块×操作维度自动计分，覆盖正确性、完整性、效率三个维度，无需人工批改
- **操作日志回放**：完整记录每位学生的每一步操作，支持按时间线回放，方便教学分析和答疑

#### 4. 六维预警系统——培养学生的风险管控意识

| 预警类型 | 触发条件 | 教学价值 |
|:---:|:---:|:---:|
| 库存不足预警 | 库存低于安全库存 | 理解库存管理与采购协同 |
| 保质期预警 | 商品接近保质期 | 理解仓储管理中的批次管理 |
| 应付逾期预警 | 应付账款超过账期 | 理解供应链金融与信用管理 |
| 应收逾期预警 | 应收账款超过账期 | 理解客户信用风险管控 |
| 运输超时预警 | 运输未按时到达 | 理解物流时效管理 |
| 合同到期预警 | 合同即将到期 | 理解合同生命周期管理 |

#### 5. 数据可视化看板——让数据"说话"

系统首页集成 ECharts 可视化图表，直观展示：
- 库存周转率趋势、采购成本对比、运输准时率、仓库利用率
- 首页仪表盘一键总览：待办事项、预警提醒、业务概况

### 五、教学价值与预期成效

**对学生的价值：**
1. **从"听懂"到"会做"**：理论课上学的采购流程、仓储管理，在系统里亲手走一遍
2. **从"个人"到"团队"**：多人协作模式培养沟通能力和团队协作意识
3. **从"静态"到"动态"**：突发事件机制培养临场应变和风险管控能力
4. **从"考试"到"能力"**：自动评分不考死记硬背，考评真实操作能力

**对教师的价值：**
1. **一键部署，开箱即用**：无需繁琐的环境配置
2. **自动评分，解放精力**：从人工批改中解放，专注教学指导
3. **实时监控，精准干预**：随时掌握每组进度，及时辅导困难学生
4. **灵活配置，千人千面**：可自定义场景、难度、评分规则，适配不同教学需求

**对专业/学院的价值：**
1. **特色教学成果**：为物流管理专业提供差异化竞争优势，可用于教学改革、专业评估、教学成果奖申报
2. **产教融合抓手**：系统对标真实企业ERP，可作为产教融合的实践载体
3. **可复制推广**：开源免费，可推广至兄弟院校，提升学校影响力

### 九、项目总结

> 本系统填补了物流管理专业"全链路仿真教学"的空白。它不是一个演示Demo，而是一个**真实可用的教学工具**——学生可以在里面完整地"经营一家物流公司"，教师可以用它高效地"管理和评估整个课堂"。
>
> 6大业务模块、5-6人协作角色、5种预设场景、6类突发事件、多维度自动评分——这不是一个作业项目，而是一个**面向教学场景的专业级仿真平台**。
>
> 技术上采用 Vue 3 + Flask + WebSocket 的主流架构，Docker一键部署，开源免费——**可部署、可推广、可持续迭代**。
